# Citation Policy

This repository is part of the open-science material for the paper **"Using AI-led semi-structured interviews to explore the connection between Carbon Tax narratives and climate anxiety"**, co-authored by **Matéo Dib, Thibaut Arpinon, and Bérangère Legendre**.

If you reuse, adapt, redistribute, teach with, or build on this pipeline, please cite:

1. this repository; and
2. the associated paper, once available.

Required repository citation:

> Dib, M. (2026). *AI-Assisted Literature Review Pipeline for Carbon and Fuel Tax Acceptability*. GitHub repository.

Recommended repository citation metadata are provided in `CITATION.cff`.

The pipeline may be adapted to other literature reviews, but the inclusion criteria, prompts, schemas, determinant families, and harmonization rules should be reviewed and modified for the new research design.

This repository does not make sensitive participant data, interview databases, copyrighted PDFs, API keys, local model caches, or raw private outputs publicly available.
